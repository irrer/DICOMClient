Quick Start How To's


Preview a series

  • Load DICOM files
  • Click Preview button to the right of the series


Preview the meta-data (text) of a series

  • Load DICOM files
  • Click Preview button to the right of the series
  • In the preview window, click the Text radio button in the lower right-hand corner of the window


Get the text of a DICOM file

  • Preview the file as text
  • Use CTRL-A and then CTRL-C to copy the text, then paste it into another application (such as Notepad).


Fix a damaged DICOM file

DICOM+ is often more tolerant of DICOM files with errors or that use older encoding methods. If you are using another application that has trouble reading a DICOM file, try this:

  • Load and preview the files in text mode
  • Click Edit to edit the file
  • Click Save+Close
The files will be saved without changing the metadata, but cleanly reformatted. Then try using the new files in the other application.


Convert DICOM files to text, XML, or PNG

  • Load and preview the files in text mode
  • Click Edit to edit the file
  • Click Options
  • Select the format(s) you want to save in
  • Click Back to Main
  • Click Save+Close


Anonymize a series:

  • Load DICOM files
  • Switch to Anonymize mode
  • Click Anonymize button to the right of the series
Generated files will be put in the Destination folder. To view them, drag them into the application and click the corresponding Preview button.
